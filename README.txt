Technologies used in the project:

Json: faster than xml and is also easily loaded in javascript.

Mockito: as the application will rely on an external database, the tests use a mock object internally created.

JUnit: to test before deploying the application.  

Postman: to test the requests between the code and the database before developing the frontend component of the application.

JUnit testing: to test before deploying the application. 

Maven: to add dependencies to the application.

Bootstrap: This is going to be used for a professional presentation at the frontend.

JQuery: easy to use
